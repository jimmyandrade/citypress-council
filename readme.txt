WordPress Theme for City Councils

== Description ==
A base ready for you to create themes for city councils. Post types and taxonomies ready to post laws, news, announcements and meetings, besides registering councilors and officials.

== Installation ==
After you install this item, you must use \"Template: citypress-council\" in style.css of your child theme.
